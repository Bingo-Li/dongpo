### 项目名：东坡
> 项目名取自唐诗《步东坡》，作者白居易  
> 苏轼是白居易的狂热粉丝，被贬黄州后，在黄州的一片山坡上耕种，自此自号"东坡居士"  
> 世间纷扰，愿每个人都找到自己的东坡

朝上东坡步，夕上东坡步。东坡何所爱？爱此新成树。  
种植当岁初，滋荣及春暮。信意取次栽，无行亦无数。  
绿阴斜景转，芳气微风度。新叶鸟下来，萎花蝶飞去。  
闲携斑竹杖，徐曳黄麻屦。欲识往来频，青芜成白路。  

### 系统简介
**目标：** 提升找房效率  
**当前系统能力：**
- 数据源：贝壳网页版本
- 数据展示方式：飞书通知、生成excel、命令行展示
- 规则配置：按照不同用户配置监控房源、配置房源黑名单
- 监控触发方式：手动触发、job触发
